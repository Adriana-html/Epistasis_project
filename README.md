# Epistasis_project
En la carpeta scripts podrá encontrar códigos de ayuda para la generación de datasets así como algunos experimentos para probar el código con diversos tamaños de dataset.

MAKEFILE
Es un archivo de ayuda para compilar y limpiar el entorno antes y después de alguna ejecución. Revisé en caso de querer correr el programa de una forma más eficiente.

SIMULDOR DE TERCERO
Para las pruebas con el simulador Epigen https://github.com/daisybio/epigen se utilizan las versiones:
mpi_epistasis_v2.cpp y serial_epistasis_v3.pp

Consideré ir al link del repositorio para poder descargar y ejecutar el simulador.
Se recomienda hacer adecuaciones en el equipo de prueba ya que este simulador utiliza corpus pesados para la generación de los datset por lo que se recomienda hacer git lfs install para poder usar los corpus# Epistasis_project
